Initial beta release of Static UI — a modern, multi-framework UI library powered by Base UI and Tailwind CSS.

- 🎨 50+ copy-paste React components (Button, Dialog, Select, Accordion, Tabs, etc.)
- 🖥️ CLI (`npx @static-ui/cli init`) for project scaffolding and component management
- 📦 Component registry with UI components, blocks, and templates
- 🔄 Multi-framework support: React, Next.js, Vue, Nuxt, SolidJS, Svelte, Astro, Remix
- 🎭 7 built-in themes: green, blue, zinc, slate, gaming, cyberpunk, modern
- 📚 Storybook documentation and interactive component playground
- 🌐 Public website with docs, live examples, and showcase
